# London-service-provision-analysis
Urban analytics project examining borough-level deprivation, libraries, and pharmacy provision across London using OpenStreetMap, IMD, and population-standardised analysis.

This repository contains the code and outputs for a borough-level urban analytics project on deprivation and everyday public service provision in London. The analysis focuses on the spatial distribution of libraries and pharmacies and examines whether borough-level deprivation is associated with service provision per 100,000 residents.

## Project focus
- Map the distribution of libraries and pharmacies across London boroughs
- Compare service provision after population standardisation
- Explore the relationship between deprivation and service provision

## Data sources
- OpenStreetMap (libraries and pharmacies)
- London borough boundaries
- Population data
- Index of Multiple Deprivation (IMD)

## Methods
- Data cleaning and preparation
- Spatial join of point data to London boroughs
- Population-standardised service rates
- Descriptive visualisation
- Correlation and simple linear regression

## Repository contents
- Jupyter Notebook with code and analysis
